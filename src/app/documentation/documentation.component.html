Introducción

Esta aplicación de demostración convierte el habla en texto utilizando tecnologías avanzadas como Whisper y Wav2Vec. A continuación, encontrarás una guía completa para utilizar la aplicación.

Uso de la Aplicación

Sigue estos pasos para convertir un archivo de audio en texto:

  1. Selecciona la tecnología que deseas utilizar (Whisper o Wav2Vec).
  2. Carga tu archivo de audio usando el botón de carga.
  3. Haz clic en "Procesar" para iniciar la transcripción.
  4. El texto transcrito se mostrará en el editor de texto.

Tecnologías Utilizadas

La aplicación utiliza las siguientes tecnologías de reconocimiento de voz:

  • Whisper: Un modelo avanzado desarrollado por OpenAI, soporta múltiples idiomas y formatos de audio.
  • Wav2Vec: Tecnología de Facebook AI, conocida por su precisión en el reconocimiento de voz.

Ejemplos de Uso

  • Subir un archivo de audio en inglés para transcripción con Wav2Vec.
